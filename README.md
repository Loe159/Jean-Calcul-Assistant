# Jean-Calcul Assistant

Assistant personnel Android open source, local-first et configurable.

Le dépôt a été réinitialisé pour repartir sur une architecture propre. Le développement commence par :

- Phase 0 : validation technique de l’intégration assistant Android/Samsung ;
- Phase 1 : assistant vocal minimal, fournisseurs interchangeables et outils Android sécurisés.

Le backlog GitHub constitue la source de vérité pour l’implémentation.
