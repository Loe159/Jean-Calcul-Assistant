# Exemples interactifs Stitch

Ce dossier conserve uniquement les deux prototypes dont le mouvement et le matériau sont difficiles à retranscrire intégralement dans une spécification statique :

- `voice-ambient.html` — orbe, respiration et état vocal focalisé ;
- `overlay-compact.html` — surface glass compacte, onde et transcription au-dessus d’une application.

Ouvrir `index.html` dans un navigateur pour accéder aux prototypes et au catalogue des autres écrans documentés dans `../SCREEN-REFERENCES.md`.

Ces fichiers servent à observer les matériaux, la hiérarchie et les animations. Ils ne définissent pas le contenu métier du produit et ne doivent pas être copiés comme code Android.

Les dépendances distantes présentes dans les exports — Google Fonts, Material Symbols et images de démonstration — sont des éléments de prototype uniquement. L’application Android ne doit pas en dépendre à l’exécution.
