# Exemples d’écran Stitch

Ces fichiers sont les exports HTML du dernier lot Stitch retenu pour l’issue #18.

Ils servent à observer la composition, les matériaux, la hiérarchie, les animations et les composants. Ils ne définissent pas le contenu métier du produit et ne doivent pas être copiés comme code Android.

Ouvrir `index.html` dans un navigateur, puis naviguer vers chaque écran.

Les dépendances distantes présentes dans les exports (Google Fonts, Material Symbols et images de démonstration) sont des éléments de prototype uniquement. L’application Android ne doit pas en dépendre à l’exécution.
